ALBAWORKは、建設業界に特化した求人マッチングアプリです。職人と企業をつなぎ、仕事探しから出退勤管理・報酬受取まで、すべてをスマホひとつで完結します。

【職人の方へ】
- エリア・日程・給与で簡単に仕事を検索
- 地図から近くの現場を発見
- ワンタップで応募、チャットで詳細確認
- QRコード+GPSで簡単出退勤
- Stripe Connect で安全・迅速に報酬を受取
- 即日払いリクエスト対応
- 品質スコアで信頼を可視化
- 資格を登録して仕事の幅を広げる

【企業・事業者の方へ】
- 写真付きの求人を簡単作成
- エリア・資格でマッチする職人にリーチ
- デジタル勤怠管理（QR+GPS）
- 日報・検査チェックリストで品質管理
- 工程タイムラインで進捗を可視化
- Stripe Connect による安全な決済

【主な機能】
- 求人検索（キーワード・エリア・日程）
- 地図検索（Google Maps連携）
- QR出退勤（GPS検証付き）
- リアルタイムチャット（画像送信対応）
- 品質スコア（評価・完了率・資格の総合評価）
- 日報作成・検査チェックリスト
- 月次明細・支払い履歴
- プッシュ通知
- 本人確認（身分証+顔写真）
- 資格管理・認証ワークフロー
- ダークモード対応
- 多言語対応基盤

【セキュリティ】
- Firebase Authentication（メール/LINE/Apple対応）
- Firebase App Check による不正リクエスト防止
- Stripe Connect による安全な決済処理
- 本人確認（KYC）機能
- 位置情報は出退勤時のみ使用（バックグラウンド追跡なし）
